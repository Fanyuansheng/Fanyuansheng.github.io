Docusaurus 教程 - 5分钟 ⏱️

便于使用

Docusaurus 是从头开始设计的，易于安装并用于让 您的网站快速启动和运行。

专注于重要的事情

Docusaurus 让您专注于您的文档，而我们会做杂 务。继续将您的文档移动到docs目录中。

由 React 驱动

通过重用 React 扩展或自定义您的网站布局。 Docusaurus 可以在重用相同的页眉和页脚的同时进 行扩展。

由 TypeScript 重构

TypeScript是JavaScript的超集,它可以编译成纯JavaScript. TypeScript可以在任何浏览器 任何计算机和任何操作系统上运行,并且是开源的.

JavaScript 框架

js框架即Javascript框架，是指以Javascript语言为基础搭建的编程框架

移动端适配

移动端适配是指在不同尺寸的手机设备上，页面能相对达到合理的 展示（响应式）或者保持统一效果的等比缩放

Nuxt.js

Nuxt.js 是一个基于 Vue.js 的通用应用框架。通过对客户端/服务端基础架构的抽象组织，Nuxt.js 主要关注的是应用的 UI渲染。

独立开发

网站是基于React的Docusaurus框架开发,服务器是基于GitHub的Pages。

项目开源

本站所有内容遵循 MIT 协议，转载须注明署名和出处，且不可用于商业用途。若与其他同步平台协议冲突，以本网站为准。
